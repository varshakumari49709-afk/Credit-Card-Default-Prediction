# Credit Card Default Prediction Project


---

# 📌 Overview

This project is a Machine Learning based Credit Card Default Prediction System developed using Python and Scikit-learn. The system predicts whether a customer will **Default** or **Not Default** on their credit card payment based on financial dataset features.

The project demonstrates the complete Machine Learning workflow including:

- Data Collection
- Data Preprocessing
- Feature Engineering
- Model Training
- Prediction System
- Accuracy Evaluation

This project was created as part of my studies at **Bakhtiyarpur College of Engineering**.

# 📈 Output

The trained model predicts whether a credit card customer will:

- **Default** (Unable to pay)
- **Not Default** (Will pay on time)

based on the input financial features (Income, Age, Loan, Loan to Income ratio).

---

# 📊 Dataset Features

The dataset consists of 2000 samples with the following features:

1. **Income** - Annual income of the customer
2. **Age** - Age of the customer
3. **Loan** - Loan amount
4. **Loan to Income** - Engineered feature (ratio of loan to income)
5. **Default** - Target variable (0 = No Default, 1 = Default)
